## Steps to create local copy:

1. fetch/clone repo
2. run npm install (note ready for yarn).
3. Add .env file with content REACT_APP_API_KEY ={your Google key}.
   4 add REACT_APP_PASSWORD ={your passwor here} key to .env file.
4. run with npm run
5. enjoy app
